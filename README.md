|#   KF5 - GIT for Arch Linux                        |
:--------------------------------------------------------------------------------:

# Download:

```
git clone https://github.com/archlinux-lucjan/kf5-git.git

```
or

```
git clone https://gitlab.com/archlinux-lucjan/kf5-git.git

```

# Install:

```
cd /some_path/kf5-git/package_name-git
makepkg -srci

```
